# Глоссарий терминов

| Термин | Определение |
|--------|-------------|
| SaaS (Software as a Service) | Модель распространения программного обеспечения, при которой продукт предоставляется как услуга и размещается у поставщика, а не на инфраструктуре заказчика. |
| On-premise | Модель развёртывания программного обеспечения, при которой продукт устанавливается и эксплуатируется на собственной инфраструктуре заказчика. |
| Парсинг (применительно к расписаниям) | Программное извлечение данных о расписании из публичных источников учебного заведения (сайт, PDF, Excel) с последующим преобразованием в структурированный формат. |
| Методист (учебного отдела) | Сотрудник вуза, ответственный за составление и поддержание актуального расписания учебных занятий. |
| Диспетчер учебного отдела | Сотрудник, отвечающий за оперативные изменения расписания (переносы, замены, изменения аудиторий). |
| HIG (Human Interface Guidelines) | Официальные дизайн-рекомендации Apple для приложений на iOS, iPadOS, macOS и других ОС Apple. |
| Material Design 3 Expressive | Текущая версия дизайн-системы Google для Android, расширяющая Material 3 более экспрессивной типографикой, движением и цветом. |
| shadcn/ui | Набор переиспользуемых компонентов на основе Radix UI и Tailwind CSS, поставляемый как исходный код для копирования в проект, а не как зависимость. |
| Liquid Glass | Визуальный язык интерфейса, введённый Apple на iOS 26. |
| Mobile-first | Подход к проектированию веб-приложений, при котором адаптация под мобильное устройство выполняется в первую очередь, а уже затем расширяется до десктопного представления. |
| MVVM (Model–View–ViewModel) | Архитектурный паттерн, разделяющий отображение, состояние представления и доменную модель. |
| MVI (Model–View–Intent) | Архитектурный паттерн, при котором взаимодействие пользователя выражается через атомарные намерения (Intent), приводящие к новому состоянию. |
| DI (Dependency Injection) | Шаблон проектирования, при котором зависимости объекта поставляются извне, а не создаются им самим. |
| SSR (Server-Side Rendering) | Серверный рендеринг HTML на сервере при каждом запросе клиента. |
| SSG (Static Site Generation) | Генерация статических HTML на этапе сборки. |
| ISR (Incremental Static Regeneration) | Гибрид SSG и SSR: статические страницы периодически пересобираются на сервере без полной пересборки сайта. |
| App Store Connect | Сервис Apple для управления распространением приложений в App Store. |
| RuStore | Магазин Android-приложений, поддерживаемый VK; основная альтернатива Google Play в России. |
| UUID (Universally Unique Identifier) | Универсальный 128-битный идентификатор, используемый в проекте для подсчёта уникальных пользователей. |
