# Личный вклад

## Распределение ролей

В проекте «ВКурсе» работают два человека:

- **Автор настоящей ВКР** — продуктовый и UX/UI-дизайн всей экосистемы; разработка всех клиентских приложений (iOS, Android, Web-лендинг, Web-приложение просмотра расписания, Web-сервис администрирования); работа с пользователями; публичная коммуникация (Telegram-канал, презентации инвесторам, переговоры с вузами). Условная роль в команде — CEO.

- **Соавтор бэкенда** — Максим, CTO. Разработка серверной части: парсинговый контур (`api.vcourse.app`) для шести омских вузов и микросервисный платформенный контур (`platform.vcourse.app`) собственного SaaS-сервиса; инфраструктура (Docker, Kubernetes, CI/CD), аналитика на Grafana. Защищает свою ВКР по платформенному контуру «ВКурсе» (парсинговый контур в его ВКР не входит).

Разделение «CEO / CTO» — условное. Юридически на момент написания работы команда не оформлена; регистрация юридического лица планируется при положительном решении по гранту Фонда содействия инновациям.

## Формулировки для введения ВКР

### Личный вклад автора

В рамках настоящей работы автором выполнены:

- Анализ предметной области и конкурентных решений.
- Продуктовое и UX/UI-проектирование всех клиентских компонентов экосистемы (мобильные приложения iOS и Android, веб-лендинг, веб-приложение просмотра расписания, веб-сервис администрирования) с раздельными Figma-файлами под каждую платформу.
- Полная реализация клиентских приложений: iOS (SwiftUI с адаптацией под iOS 18− и iOS 26+ с Liquid Glass), Android (Jetpack Compose, Material Design 3 Expressive), Web (Next.js, Tailwind CSS, shadcn/ui).
- Подготовка и проведение релизов в App Store, Google Play и RuStore.
- Сбор пользовательской обратной связи, поддержка Telegram-канала проекта.
- Организационная работа: переписка с учебными заведениями, выступления перед инвесторами, презентация на ОмГАУ.

### Разграничение со смежной работой

Серверная часть платформы «ВКурсе» — парсинговый контур для шести омских вузов и микросервисный платформенный контур собственного SaaS-сервиса, а также связанная инфраструктура — разработана соавтором. Платформенный контур является предметом отдельной выпускной квалификационной работы соавтора. В настоящей ВКР обе серверные подсистемы рассматриваются только в объёме интеграционного контракта, необходимого для описания клиентской составляющей.
