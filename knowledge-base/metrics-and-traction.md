# Метрики использования

Снимок данных по состоянию на середину мая 2026.

## Установки и активность

- **Суммарно установок мобильных приложений:** ~3 000.
- **Среднее число уникальных активных пользователей в день (на мобильных):** ~1 000.

## Распределение по платформам

| Платформа | Доля пользователей |
|-----------|---------------------|
| iOS | 79% |
| Android | 19% |
| Web | 1% |

Доминирование iOS объясняется историей: приложение для iOS вышло первым (сентябрь 2025), Android и Web появились только в феврале 2026 года. Распределение постепенно выравнивается.
