# Web-сервис администрирования

## Известное на сейчас

- **Размещение:** на том же домене, что и публичный веб (лендинг + расписание), но доступно только после авторизации.
- **Аудитория:** администраторы, диспетчеры, методисты учебных заведений.
- **Стек:** общий с публичной частью — Next.js, Tailwind CSS, shadcn/ui.
- **Статус:** в активной разработке, дорабатывается к моменту защиты ВКР.

## Функциональность

### Авторизация

Закрытый доступ. Два уровня пользователей в рамках одного учебного заведения:

- **Администратор** — может добавлять и удалять сотрудников, изменять данные заведения (название, логотип и т.д.), плюс всё, что может сотрудник.
- **Сотрудник** — методист / диспетчер с правами на работу с расписанием и справочниками.

### Управление справочниками

- Группы.
- Преподаватели.
- Аудитории.
- Факультеты.
- Кафедры.
- Корпуса.

### Управление сотрудниками

- Список сотрудников.
- Добавление и удаление учётных записей.
- Доступно только администратору.

### Составление расписания

- Сетка слотов.
- Внесение изменений с проверкой коллизий (на стороне бэка).
- (Эта функция — основная разрабатываемая в момент написания ВКР.)
