# База знаний по ВКР «ВКурсе»

Это рабочая директория для подготовки ВКР. Все ключевые факты о проекте — продукт, история, конкуренты, метрики, апробация — собраны здесь в виде атомарных `.md` файлов. При написании любой главы ВКР сначала следует сверяться с этой базой.

## Как этим пользоваться

- Каждый файл — одна тема. Не дублируем факты между файлами; вместо этого ставим перекрёстную ссылку.
- Если факт изменился (например, выросло число пользователей) — обновляем **только в одном месте**.
- Файлы `clients/*.md` пока неполны: их детали будут уточнены при отдельной сессии ревью репозиториев iOS/Android/Web.
- `pending-questions-backend.md` — список вопросов для Максима (соавтор бэка). После ответов перенести факты в `backend-contract.md`.
